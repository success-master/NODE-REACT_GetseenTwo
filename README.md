# getseen-two
https://getseen.co.uk

## run getseen locally

Clone respository. 
cd into project root directory.

$ cd getseen-two

### Run client (default: localhost:3000) 
$ cd client
$ npm install
$ npm start

### Run server (default: localhost:4242)
$ cd server
$ npm install 
$ npm run server
